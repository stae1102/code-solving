# Tree

[메인으로 돌아가기](../../README.md)

트리는 자료의 구조를 계층적으로 나타내는 구조이다. 특히, 트리는 이진트리 중 완전이진트리가 중요하다. 기존 탐색에 소비되는 시간복잡도가 O(N)에서 O(logN)으로 감소하기 때문이다.

완전이진트리는 데이터베이스 프로그램에서 Btree를 사용하고 HTML에서도 DomTree를 사용한다.
